Frequently Asked Questions

Everything you need to know about our free tools

Is ScrapeStack really free?
Yes! ScrapeStack is completely free. No premium tiers, no hidden costs, no credit card required. We believe essential tools should be accessible to everyone.
Do you add watermarks to converted files?
Never. All converted files are clean and watermark-free. We don't add any branding to your documents or images.
Is my data private and secure?
Absolutely. Image compression happens entirely in your browser - files never leave your device. For document conversion, files are automatically deleted after 1 hour.
What file formats do you support?
Images: JPG, JPEG, PNG, WEBP, GIF, BMP. Documents: DOC, DOCX, PDF, PPT, PPTX. Output formats include PDF, optimized images, and merged PDFs.
Is there a file size limit?
Image compression has no size limit (works directly in browser). Document conversion supports up to 100MB per file to ensure reasonable processing times.
